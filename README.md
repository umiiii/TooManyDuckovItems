# 鸭科夫物品获取方式分析 Mod

## 功能说明

这个 Mod 会在启动时自动分析并打印游戏中所有物品的获取方式，帮助你了解：

- 哪些物品可以通过制作获得
- 哪些物品在商店出售
- 哪些物品是任务奖励
- 哪些物品可以通过种植获得
- 哪些物品是任务专属生成

## 安装方法

1. 编译项目生成 `DisplayItemValue.dll`
2. 将 DLL 文件放到游戏的 Mods 文件夹中
3. 启动游戏

## 版本历史

- v1.0 - 初始版本
  - 支持制作、商店、任务奖励、种植系统分析
  - 自动在启动时打印所有物品获取方式
